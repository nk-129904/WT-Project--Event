# EventEase — Personal Event Manager (Dark Theme)

## What is this
EventEase is a lightweight client-side event/reminder manager.  
Built with HTML, CSS, Bootstrap, Tailwind, and Vanilla JavaScript. Data stored in browser localStorage.

## Pages
- index.html — landing/home
- create.html — add events
- view.html — view events (cards + pagination)
- edit.html — edit event (client)
- about.html — about
- admin/* — admin pages: dashboard, manage, edit, delete, logs

## How to run
Open `index.html` in your browser. No server required.

## Notes
- All images removed — no 404 errors.
- Modals, offcanvas, alerts, pagination are implemented using Bootstrap & JS.
- Validation: title and date required; date must be today or future.
